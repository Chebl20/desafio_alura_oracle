Desafio Codificador Texto

O SHA-256 é um algoritmo de hash criptográfico que é unidirecional. Isso significa que uma vez que os dados são convertidos em um hash SHA-256, não é possível reverter o hash para recuperar o texto original. Portanto, a descriptografia do hash SHA-256 não é possível.
